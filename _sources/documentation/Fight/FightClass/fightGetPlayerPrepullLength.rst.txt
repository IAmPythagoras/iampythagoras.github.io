Fight.getPlayerPrepullLength()
==============================

Computes and returns the length of the prepull of all players. Returns a dictionnary that maps playerID to prepull length.

**Return**

A dictionnary where the key is a player's id (int) and it maps to the length in seconds of its prepull (float).

**Usage**

.. code-block:: python  

    newFight.getPlayerPrepullLength()
