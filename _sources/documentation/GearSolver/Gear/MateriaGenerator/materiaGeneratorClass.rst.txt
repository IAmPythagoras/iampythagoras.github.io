MateriaGenerator
================

This class is used to generate materias given certain parameters.

**Class attributes**

* MateriaGenerator.OddValue : int - Value of the generated materia when the tier is odd.
* MateriaGenerator.EvenValue : int - Value of the generated materia when the tier is even.
