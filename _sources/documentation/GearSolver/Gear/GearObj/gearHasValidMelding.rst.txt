Gear.hasValidMelding()
======================

This function returns True if the gear has a valid melding.

**Return**

This function returns a boolean.

**Usage**

.. code-block:: python

    illegalGearPiece.hasValidMelding() # Would return false if the gear is illegaly melded
    legalGearPiece.hasValidMelding() # Would return true if the gear is legally melded.
