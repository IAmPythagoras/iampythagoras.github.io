Gear.canReplaceMateriaNoLoss()
==============================

This function returns True if a materia of the gear piece can be replaced by a new materia without stat loss.

**Parameter**

* newMateria : Materia - Materia to check replacement for.

**Return**

This function returns a boolean.

**Usage**

.. code-block:: python

    newGear.canReplaceMateriaNoLoss(newMateria)
