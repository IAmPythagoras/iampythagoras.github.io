Gear.__str__()
==============

This function casts a Gear object into a string.

**Returns**

This function returns a string of the format : 'Gear.GearType : Stat1 : Stat1.Value | Stat2 : Stat2.Value | Stat3 : Stat3.Value |  Materias : Mat1 : Mat1.Value | Mat2 : mat2.Value |  Name : Gear.Name'.

**Usage**

.. code-block:: python

    print(newGear)
    # would print out something like that :
    # HEAD : SS : 184 | Det : 129 | MainStat : 248 |  Materias : DH : 36 | DH : 36 |  Name : Raid
