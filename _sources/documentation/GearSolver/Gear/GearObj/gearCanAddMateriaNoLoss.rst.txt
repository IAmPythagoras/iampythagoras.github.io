Gear.canAddMateriaNoLoss()
==========================

This function returns weither the materia can be added to the gear piece without stat loss.

**Parameter**

* newMateria : Materia - Materia to check for.

**Return**

This function returns True if the materia can be added without loss and False otherwise (boolean).

**Usage**

.. code-block:: python

    newGear.canAddMateriaNoLoss(newMateria)
