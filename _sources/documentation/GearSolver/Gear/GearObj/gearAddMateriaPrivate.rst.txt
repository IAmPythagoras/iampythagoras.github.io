Gear.__addMateria()
===================

.. note::

    This is a private function that can only be called within the Gear class.

This function contains the inner logic of how to add materias to the gear. This function is called by
Gear.AddMateria and Gear.ForceAddMateria.

**Parameter**

* newMateria : Materia - Materia object to add.

**Usage**

.. code-block:: python

    newGear.__addMateria(newMateria)
