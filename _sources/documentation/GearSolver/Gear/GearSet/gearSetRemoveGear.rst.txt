GearSet.RemoveGear()
====================

This function removes the given GearType from the GearSet.GearSet dictionnary.

**Parameter**

* Type : GearType - Type of the gear to remove.

.. note::

    If not Gear matches the Gear type nothing is done.

**Usage**

.. code-block:: python

    newGearSet.Remove(GearType.WEAPON)
