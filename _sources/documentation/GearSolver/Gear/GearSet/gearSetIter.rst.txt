GearSet.__iter__()
==================

This initializes the GearSet iterator. It initializes the value of GearSet.gearSetIter which is simply an iter object on GearSet.GearSet dict.

The iterator iterates over the GearSet and returns the Gear objects that are part of the GearSet.

**Return**

This returns the object on which it was called (GearSet).

**Usage**

.. code-block:: python

    iter(newGearSet)
