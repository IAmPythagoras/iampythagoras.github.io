Jobs
====

Jobs is a whole module of **ffxivcalc**. It contains the implementation of all thing
related to a Player. It is one of the most (if not the most) important part of this whole library.
It contains multiple files and class that are all of great importance.

**Import**

.. code-block:: python

    from ffxivcalc.Jobs import *
