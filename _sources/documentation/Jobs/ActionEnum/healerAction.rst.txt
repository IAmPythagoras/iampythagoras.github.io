HealerAction 
============

This is a subclass of ActionEnum. HealerAction is an enum mapping the name of actions
that healers can do to their unique id.

**Enum**

* Repose = 16560
* Esuna = 7568
* Rescue = 7571
* Swiftcast = 7561
* LucidDreaming = 7562
* Surecast = 7559
* Potion = -2
* WaitAbility = 212
* HealerLB1 = 1111
* HealerLB2 = 1112
* HealerLB3 = 1113
