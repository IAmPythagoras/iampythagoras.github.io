MeleeAction 
============

This is a subclass of ActionEnum. MeleeAction is an enum mapping the name of actions
that melees can do to their unique id.

**Enum**

* SecondWind = 7541
* LegSweep = 7863
* Bloodbath = 7542
* Feint = 7549
* TrueNorth = 7546
* ArmLength = 7548
* Potion = -2
* WaitAbility = 212
* AA = -22
* MeleeLB1 = 1111
* MeleeLB2 = 1112
* MeleeLB3 = 1113
