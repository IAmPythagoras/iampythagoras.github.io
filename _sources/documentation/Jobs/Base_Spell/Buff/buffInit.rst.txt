buff.__init__()
===============

This is the constructor of the buff class.

**Parameters**

* MultDPS : float - Value of the multiplicative buff.
* name : str - Name of the buff. (Default "Unnamed")
* isDebuff : bool - True if the buff is a debuff. (Default False)

**Usage**

.. code-block:: python

    newBuff = buff(1.1, "FunnyBuff", isDebuff = False)
