Player.addBeastChakra()
==========================

.. warning::

    This function only works if the player has been initialized using *init_monk()*.

This functions adds a beast chakra type to the player's master gauge.

**Parameter**

* type : int - Type of the chakra beast (1 - bootshine type, 2- true strike type, 3- demolish type)

**Usage**

.. code-block:: python

    mnkPlayer.addBeastChakra(1)
